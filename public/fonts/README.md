# Carpeta de Fuentes
Coloca aquí tus archivos de fuentes ( .ttf, .otf, .woff2 ).

Para usarlas en la aplicación, debes registrarlas en `src/index.css` usando `@font-face`.

Ejemplo:
```css
@font-face {
  font-family: 'MiFuentePersonalizada';
  src: url('/fonts/mi-fuente.ttf') format('truetype');
  font-weight: normal;
  font-style: normal;
}
```
